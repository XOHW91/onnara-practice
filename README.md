온나라 공문 작성 실습 (Onnara Document Practice)


신입 공무원이 실제 업무에 투입되기 전, 공문서 작성 규칙과 온나라 시스템 UI를 미리 손에 익힐 수 있도록 만든 웹 기반 실습 프로그램입니다.



왜 만들었나요

신입으로 처음 발령받으면 공문 하나 쓰는 것도 막막할 때가 많습니다. 날짜는 어떻게 쓰는지, "끝." 은 어디에 어떻게 붙이는지, 붙임은 띄어쓰기를 어떻게 해야 하는지 등 사소해 보이지만 틀리면 눈에 띄는 규칙들이 은근히 많습니다.

이 프로그램은 실제 온나라 시스템 화면을 최대한 비슷하게 재현해서, 실무 문서를 실제로 발송하기 전에 미리 연습해볼 수 있도록 만들었습니다. 저 역시 신입으로 입직하면서 같은 답답함을 느껴 직접 만들게 되었습니다.

무엇을 연습할 수 있나요
